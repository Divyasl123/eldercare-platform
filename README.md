# 🏥 Elderly Healthcare Platform

A Full Stack Web Application for managing elderly healthcare services.

---

## 📌 Features

- User Registration & Login
- Admin Login
- Patient Profile Management
- Service Booking
- Caregiver Management
- MongoDB Database Integration

---

## 🛠 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

---

## ⚙ Installation Steps

1. Clone the repository:
   git clone https://github.com/Divyasl123/eldercare-platform.git

2. Go to project folder:
   cd eldercare-platform

3. Install dependencies:
   npm install

4. Go to server folder:
   cd server

5. Start server:
   node server.js

Server runs on:
http://localhost:5000
